# Inteligencia Computacional

Este repositorio contiene tareas, prácticas y proyectos desarrollados durante la materia de **Inteligencia Computacional**. Aquí se irán subiendo todos los trabajos realizados a lo largo del curso como respaldo y seguimiento del aprendizaje.

## Contenido del repositorio

Actualmente, el repositorio incluye los siguientes archivos y carpetas:

- **Feed-Forward-Network/**  
  Implementación de una red neuronal feed-forward, incluyendo código y experimentos relacionados.

- **Flight-Price-Prediction/**  
  Proyecto enfocado en la predicción de precios de vuelos utilizando técnicas de machine learning.

- **Modelo-ML/**  
  Carpeta con actividades relacionadas al desarrollo y uso de modelos de aprendizaje automático.

- **EDA_y_Preprocesamiento.ipynb**  
  Notebook que contiene análisis exploratorio de datos (EDA) y técnicas de preprocesamiento.

- **Metricas_y_Evaluacion.ipynb**  
  Notebook donde se analizan métricas de evaluación para modelos de machine learning.

## Notas

- Este repositorio está en constante actualización conforme se avanzan los temas de la materia.
- Los archivos están organizados por tema o proyecto para facilitar su comprensión.
